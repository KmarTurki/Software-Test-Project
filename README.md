## Linking to GitHub
To link this local repository to a new GitHub repository:

1.  Create a new repository on GitHub (do not initialize with README, .gitignore, or License).
2.  Run the following commands in your terminal:

```bash
git remote add origin <REMOTE_URL>
git branch -M main
git push -u origin main
```

Replace `<REMOTE_URL>` with the URL of your new GitHub repository (e.g., `https://github.com/yourusername/software-test-project.git`).
